Ohmyzsh Command Line Highlight Tool

1 min read.

Ohmyzsh is a commandline highlight tool #

https://github.com/ohmyzsh/ohmyzsh

Install at Ubuntu #

  1. Install zsh first

    $ apt-get install zsh

  2. Clone Ohmyzsh

    $ git clone https://github.com/ohmyzsh/ohmyzsh.git

  3. Install Ohmyzsh

    $ cd /ohmyzsh/tools/
    $ ./install.sh

  4. Change the setting

    $ nano ~/.zshrc
    ZSH_THEME="agnoster"
